部门介绍

“聚是一团火，散是满天星”，应用开发部，作为科协技术中心三大部门之一，是一个充满活力的部门。“聚”在一起，我们可以一同探究设计与开发，在思维的碰撞和一行行代码里见证产品的落地；“散”开之后，我们深耕各自方向，在代码里探寻优雅与设计，在多媒体中探寻科技与创新。 你可以在视觉组学习平面与UI的设计；在媒体组学习炫酷的特效与动画；在摄影组体验光影的艺术；在音频组谛听音乐的灵动；在游戏组享受使用 Unity 引擎开发的乐趣。你也可以在前端组和后端组将一个个设想，在清晰优雅的代码块中实现，亲身经历一个项目从无到有，从0到1的流程。你也可以在运维组手握八方服务器，洞察云计算的奥秘。

从前端到运维，从音频到摄影，Web开发和多媒体设计是部门的两大主线，我们会结合具体项目与应用，同学弟学妹一道，迈过从零到一的槛，在项目驱动中提升自己的水平。


前端组

前端是展示个人想法、个人审美、个人设计的工具，是提高艺术欣赏能力和代码书写能力的道路，也是了解计算机网络，了解不同形式的代码，了 解不同方式的展示，和发挥个人想象能力的途径。

前端不只是基础静态网页的三大基本语言: HTML5、CSS、JavaScript的学习，更有着动态的响应事件、数据库的接入以及许多现在流行的框架，如：Vue、React、Angular 等等。“丹山九仞，烟峰非数篑之功； 紫极千门， 云台俟万楹之力。”前端的知识将循序渐进，不断丰富，提高你的知识储备量，直到你可以实现你的美，你的艺术。

前端也是科协产出大户，很多讲师都曾参与到，诸如：南邮小程序、NOJ 在线评测系统、新柚杯比赛系统、AtSAST 等项目的开发中去。

总而言之，前端是一个充满挑战和乐趣的方向，我们期待在攀登前端的山峰时，能够与你一起，欣赏大前端的瑰丽。


后端组

后端开发工作是对你们算法能力、业务逻辑的展示，在一次次的头脑风暴中扩展自己的思维。后端主要负责对页面中传来的数据进行处理，针对程序设计架构思想，通过连接数据库进行增删改查等一系列基本操作。在后端，我们希望可以依靠自己的逻辑思维，将一些本身枯燥的东西变得简单而优雅。它，并不是简单的 CRUD 。

比起前端来说，后端更像是一个挑战。这并不是说前端开发人员不解决难题，但通常前端开发工作更多的是关于创建用户界面和与之相关的内容，而不是实现实际的业务逻辑，使应用程序工作。校科协也有很多从后端出来的优秀前辈，他们都有着炉火纯青的代码能力，都是后端组一个个等待去超越的高峰。不仅如此，每年举办的新柚杯以及即将问世的 ATSAST 等项目，都是后端稳定的项目产出。

本学年我们主要会联合 Java 组进行联合授课，在后面的阶段，我们也会去摸索“后起之秀”—— Go 语言。如果你喜欢设计应用程序的基础设施，实现算法和逻辑，以及工作于数据，那么欢迎来到后端组，成为一名出色的后端开发人员。


运维组（Linux组）

想体验手指只在键盘上跳跃的急速闪电吗？还在为win动不动黑屏，再动不动蓝屏而感到困扰吗？欢迎来运维组摸鱼。运维组主要还是以Linux相关教学为主（所谓命令行才是程序猿的浪漫），此外我们会介绍数据库，Git（版本控制），Github（交友网站），以及Docker（应用容器）。总之，欢迎来玩，来交流。


媒体组&视觉组&音频组

我们主要负责应用开发部的产品美术、用户体验与交互设计等，以及产品的媒体宣传。

授课内容主要包含但不限于平面设计、UI 设计、音频制作、摄影技术、影视剪辑后期、MG 动画、Blender 建模、新媒体传播等等，大家共同在多媒体的世界探索，我们讨论氛围浓厚，可以在这里畅所欲言，表达自己的设计制作想法，合适的时候也会经常举办大型美学沙龙。

我们的前辈曾参与 B 站拜年祭、明日方舟等项目的制作，沙龙很多时候就是“老带新”的重要窗口。现任成员也有着较为丰富的影视后期剪辑、音频制作、影片拍摄经验。P.S. 今年的科协宣传视频会在招新期间放出，尽请期待哦~


摄影组

“摄影的意义，简单说就是用相机记录历史，让历史定格在一瞬间，让一瞬间成为永恒。“摄影组作为多媒体，一直承担着教授大家摄影相关知识技巧和培养有关美感的任务。

这学年摄影组的主要教授课程大致可以分为：摄影有关历史文化知识；设备使用基础知识和技巧；各类摄影的各自特点及过程技巧；平面后期的相关知识（ Lr 和 Ps 以及 APP）

两位讲师并非全栈摄影师，一号讲师较为擅长新闻摄影，纪实摄影，风光摄影和 Lightroom 的使用，还比较熟悉无人机摄影的一些技法；二号则讲师较为擅长 Photoshop 后期修图及建筑与静物摄影，对色调搭配有自己的见解。

还希望能和大家探讨学习我们不很擅长的领域，一同学习进步。


游戏组

“同学，玩了这么多游戏，你有没有想过做一款属于你自己的游戏呢？你知道游戏是怎么设计开发的吗？” 在游戏组，我们会教授你“游戏设计圣经”，从设计出发，思考如何做出一款让众人拍手叫好的游戏，引导你一步一步成长为优秀的游戏设计大师。

不止于游戏设计，爱玩游戏的你一定听说过炉石传说、原神、城市天际线、神庙逃亡、逃离塔科夫等热门游戏，这些游戏的共同点是，它们都基于 Unity 引擎进行开发，而 Unity 引擎，也是目前市场占有率最高的游戏引擎。在游戏组，我们也将从 Unity 开始，带你进入有趣的游戏开发世界。
