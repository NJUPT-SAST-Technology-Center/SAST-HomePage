部门介绍

软件技术部技术方向众多，对于技术的追求的氛围是整个部门的主基调，浓厚的技术氛围带来的不仅仅是技术的无条件分享，更重要的是自由的浪漫。在 Java 组，你将用 Java 学习到后端开发并体会到 Java 程序的优雅；在 Python 组，你将走进人工智能与机器学习，制作属于自己的 AI；在 C# 组，你可以学习 .NET 平台 UI 设计，接触跨平台开发的新技术，体验 C# 精美的语法与不俗的性能；在 C++ 组，你将走进规范、现代的 C++ 新标准，并上手 Qt 桌面应用开发；在 Lambda 组，你能领会函数式编程之美，学习真正的计算机科学；在 Graphics 组，你将学习到计算机图形处理的巧妙，在计算机中创造光影；在算法组，你会从算法和数据结构中解出人生真谛；在安全组，你将走进攻防世界，传承黑客精神。


Java组

Java 是一门简单的面向对象的程序设计语言。语法与 C/C++ 接近，易于学习和使用。浙江大学教授翁恺曾经这样评价过 Java：“Java 使人内心宁静。”在学习 Java 的过程中，其很多特性都在给人透露出一种优雅与美。

Java 社区庞大，生命力强。在编程语言界，尽管有许多后来之秀，但 Java 仍然占据并且在可预见的未来也会长期占据着 TIOBE 排行榜前列，足以证明这是一门实用且应用广泛的语言。目前 Java 语言在后端开发方向应用较多，本学期 Java 组也将与校科协后端组以及计软院科协后端组联合授课。


Graphics组

告诉你个小秘密：“游戏场景越亮，游戏越好”，至于原因嘛，来听课便知~

因为我们将在 Graphics 组开展 Computer Graphics （计算机图形学）相关的授课与讨论，而它的运用，遍及电影、游戏、VR、动画、设计、AR等等。我们将着重介绍 Rendering（渲染技术）方向，带领大家将输入计算机的数据转化为图形。到那时，你可随心所欲地在计算机中创造光影，用代码为摆设的物体打上高光，漫反射与阴影，物体的材质，是金属还是塑料，全都由你决定！


C++组

​ C++ 是一门具有高度自由性的语言。最开始，它只是为了迎合日渐流行的 OOP 的观点，被作为 C with classes 开发的，而后却变得无比庞大，以至于 C 都成了 C++ 的一个小小的子集。

不同的人对 C++ 有着不同的理解，ACM 选手或许会认为 C++ 是 C with STL，而图形学者则可以使用 C++ 的 OpenGL 等库。而我们 C++ 组的目的是使用 C++ 开发出能够解决实际生活问题的，方便的程序。


C# 组

​C# 是微软公司开发的一款简单、现代、面向对象的编程语言，基于 .NET 环境运行。近年来，随着微软与开源社区的共同努力，.NET 在越来越多的平台上运行。无论是桌面端、移动端还是 Docker 容器，都能看到 .NET 的身影。

C# 在语法和特性上博采众长。既有 C++ 的丰富成员支持，又有 Java 简洁的对象结构与自动垃圾回收机制。不仅如此，C# 的委托、事件与 LINQ 也是经典之作。大量语法的加持并没有使 C# 变得臃肿，而是更加灵活，充满美感。尽管在 TIOBE 长期处于第五位，C# 依然拥有着巨大的潜能，每次更新都会让人眼前一亮。

往届校科协 C# 组主要研究基于 Unity 的游戏开发，但是今年，我们决定探索 C# 的更多可能：桌面开发、UI 设计、数据处理、网页交互、移动开发……欢迎各位同学和讲师一起，发掘 C# 语言的无限可能。


Python组

至于Python是什么想必大家都不陌生啦，各位小伙伴可能早就已经听过啦，这里我着重介绍一下我们组将会用 Python进行什么实战，我们将会从如下几个方面对大家进行教学：

计算机视觉：

大家在我们部门能够学习到计算机是如何通过“眼睛”去理解世界的，了解计算机如何去感知图像，从感知中如何提取信息，并且利用提取出来的信息来为我们人类去服务。

自然语言处理：

讲完计算机如何去看世界，这时候我们还得学计算机如何“讲话”，能够语言本身只有生物才能拥有，但是对于聪明的人类来说，计算机也能够听懂人类的话，也能够和人类进行语言上的交流，早在 2015 年就已经有机器通过图灵测试成功让人类相信它是一个 13 岁的男孩。

强化学习：

讲完说话和看世界，得让机器人真正的具有智能还得看强化学习，强化学习也是目前让 Alpha Zero 从零到有就能以 100：0 的战绩战胜 Alpha go 的唯一方法，我们只需将世界的规则输入给计算机，然后让计算机自己去摸索，“不一会儿”它就能在各行各业超过人类，单手玩魔方，玩星际争霸 2，通过人类闻所未闻的策略让人类玩家输的心服口服，在各种游戏里面的实际效果早已超越人类玩家，甚至给人类玩家提供了游戏思路。

当然还有很多的方方面面，这里就不一一总结了，欢迎各位同学和讲师一起对人工智能进行探索和研究。我们相信人工智能的时代将会来临。


安全组

学习安全知识，走入牢狱人生，走入秃头人生。安全组学习方向现在大体分为二进制漏洞利用、web攻防技术、密码学方向、逆向分析技术与Misc，不同的学习方向侧重点不同。

二进制漏洞利用（PWN）：

二进制漏洞利用方向又可以称为pwn方向。何为pwn方向呢。简而言之就是软件漏洞挖掘，挖掘软件漏洞，并且对之进行利用，如果该软件正好运行在别人电脑或者服务器上那么你便可以利用该漏洞取得最高控制权限进而控制别人电脑或者服务器。（是不是感觉很像电影中的黑客们呢？）目前，pwn方向是偏向于打CTF竞赛，当前也有玩得溜的学长们走上了真正的安全研究岗位，真实的漏洞发掘，成为保卫企业安全乃至国家安全的力量。

Web攻防技术（Web）：

web方向的学习不同于二进制需要对计算机系统深入的理解，很多大师傅也是半路出家，凭借自己的一手“奇淫技巧”，成为了“一代宗师”。二进制安全和web安全就像武侠小说中的“气宗”和“剑宗”，各有各的门道。正因为web安全不需要非常深厚的代码功底，很多CTFer认为web安全是最容易入门的。但是这个“入门”并不是真正意义上的入门，也许连web安全中的冰山一角都算不上。web安全知识很零碎，在web安全学习过程中要注重积累，形成自己的知识库。

密码学(Crypto)：

密码学顾名思义就是研究加密的学科。比如要在Alice与Bob两人通信过程中，在有Eve窃听的情况下，依然保证消息不泄露，这就需要Alice用一个加密密钥（类似于开锁的钥匙）对信息加密，而Bob将收到的信息用解密密钥解密，这样Eve就无法得知通信内容。由于现代密码学要求在加密算法公开的情况下，只要不知道密钥，就无法对消息进行解密。这样的话，仅需要保护一个不算长的密钥即可保护一段信息；即使密钥泄露，换个密钥就能继续用同一个加密算法加密。所以，密码学是要寻找一个在不知道密钥情况下寻找算法漏洞进而破解加密信息的学科。

逆向分析技术(Reverse)：

当Cynosure写好了一个有趣的游戏正想分享给他的好朋友Kagehutatsu一起娱乐时，或许是害怕自己游戏中的核心代码（游戏设计BUG）被好朋友通过逆向的手段发现，也或许是害怕好朋友根据其中的特性 设计漏洞 编写出特定外挂来破坏游戏的平衡，Cynosure决定对自己打包的游戏本体加上各种诸如软件加壳，虚拟机设计，双进程保护以及反调试手段来对自己的游戏进行保护。而这些保护手段就是一个逆向分析人员需要面对的东西。逆向分析方向侧重于对软件代码的分析，包括软件的破解，恶意代码的分析，在代码分析的历程中感受软件保护技术的防护与攻击，通过持之以恒的努力，最终对软件进行破解后的成功与兴奋，会是你在这条路上越走越远坚持的动力。

Misc:

Misc是英文 Miscellaneous 的前四个字母，杂项、混合体、大杂烩的意思。隐写取证是 Misc 中最为重要的一块，包括文件分析、隐写、内存镜像分析和流量抓包分析等等，涉及巧妙的编码、隐藏数据、层层嵌套的文件中的文件，灵活利用搜索引擎获取所需要的信息等等。CTF 中 Misc 与现实中的取证不同，现实中的取证很少会涉及巧妙的编码加密，数据隐藏，被分散嵌套在各处的文件字符串，或是其他脑洞类的 Challenge。Misc 是切入 CTF 竞赛领域、培养兴趣的最佳入口。
